Interactive Power BI dashboard for Product Sales &amp; Shipment Analysis featuring KPIs, sales trends, country-wise performance, product analysis, shipment tracking, and salesperson insights. Built using Power BI, Power Query, DAX, and Excel to support data-driven business decisions.
# 📊 Product Sales and Shipment Analysis Dashboard

## 📌 Project Overview

The **Product Sales and Shipment Analysis Dashboard** is an interactive **Power BI** project designed to analyze sales performance, shipment metrics, product trends, country-wise performance, and salesperson productivity. The dashboard provides meaningful business insights through interactive visualizations, helping organizations monitor KPIs and make data-driven decisions.

---

## 🎯 Project Objectives

- Analyze overall sales performance.
- Monitor product shipment statistics.
- Identify top-performing products.
- Evaluate salesperson performance.
- Compare sales across different countries.
- Track sales trends over time.
- Build an interactive business intelligence dashboard.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel (CSV Dataset)

---

## 📊 Dashboard Features

- 💰 Total Sales KPI
- 📦 Total Orders
- 🚚 Average Shipment Boxes
- 📈 Total Boxes Shipped
- 🌍 Country-wise Sales Analysis
- 📦 Country-wise Shipment Analysis
- 👨‍💼 Salesperson Performance Analysis
- 🍫 Product-wise Sales Analysis
- 📅 Yearly, Quarterly, Monthly & Daily Sales Trends
- 🎛️ Interactive Filters and Slicers
